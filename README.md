# Sistema de Controle de Estoque, Caixa e Vendas

Sistema desktop desenvolvido em julho de 2025 para a disciplina de Engenharia de Software II durante o 6º semestre da graduação.

## 📌 Descrição

O projeto consiste em um software de gerenciamento comercial desenvolvido em Python com interface gráfica, voltado para o controle de estoque, vendas e caixa.

O sistema permite o gerenciamento de produtos, controle de quantidade em estoque, registro de vendas, cálculo de lucro e acompanhamento do fluxo comercial de maneira simples e organizada.

---

## 🚀 Funcionalidades

* Cadastro de produtos
* Controle de estoque
* Atualização automática de quantidade
* Registro de vendas
* Carrinho de compras
* Controle de caixa
* Histórico de vendas
* Cálculo de lucro
* Alertas de estoque baixo
* Alertas de validade próxima
* Controle de fornecedores
* Interface gráfica desktop

---

## 🛠️ Tecnologias Utilizadas

* Python 3.11
* Tkinter
* Pandas
* NumPy
* Tkcalendar
* CSV para armazenamento de dados
* PyInstaller

---

## 📚 Objetivo Acadêmico

Projeto desenvolvido como atividade acadêmica da disciplina de Engenharia de Software II, com foco na aplicação prática de conceitos de desenvolvimento de software, organização de sistemas e gerenciamento de dados.

---

## ▶️ Como Executar

1. Instale as dependências:

```bash id="ft96qj"
pip install pandas numpy tkcalendar
```

2. Execute o arquivo principal:

```bash id="z0x9w5"
python estoque.py
```
